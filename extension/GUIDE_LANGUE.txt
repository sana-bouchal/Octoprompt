┌───────────────────────────────────────────────────────────────┐
│  ÉTAPE 1 : Ouvrir le Popup                                    │
└───────────────────────────────────────────────────────────────┘

   1. Cliquez sur l'icône 🐙 OctoPrompt dans la barre d'extensions
   2. Le popup s'ouvre avec l'interface

┌───────────────────────────────────────────────────────────────┐
│  ÉTAPE 2 : Localiser le Toggle de Langue                     │
└───────────────────────────────────────────────────────────────┘

   Vous verrez une ligne avec :
   
   📌 Langue: Français  [  ○─────  ]
   
   ou
   
   📌 Language: English  [  ─────○  ]

┌───────────────────────────────────────────────────────────────┐
│  ÉTAPE 3 : Changer la Langue                                 │
└───────────────────────────────────────────────────────────────┘

   • Toggle à GAUCHE (○─────) = Français 🇫🇷
   • Toggle à DROITE (─────○) = Anglais 🇬🇧
   
   → Cliquez sur le toggle pour basculer
